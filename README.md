# 🛰️ AI-Радар — Telegram Mini App

Мини-апка с новостями нейросетей: карточки с русскими пересказами, лайки/дизлайки, рейтинг «🔥 Топ». Лайки общие для всех пользователей и хранятся на сервере.

## Что внутри

- `server.js` — сервер: отдаёт апку, хранит лайки, принимает свежие новости
- `public/index.html` — сама мини-апка
- `news.json` — стартовый набор новостей (потом обновляется автоматически)
- `package.json` — зависимости

## Настройка (~15 минут)

### Шаг 1. Залей код на GitHub

1. Создай новый репозиторий на github.com (можно приватный), например `ai-radar-miniapp`
2. Загрузи туда все файлы из этой папки (кнопка «Add file → Upload files» — перетащи файлы и папку public)

### Шаг 2. Разверни на Railway

1. В Railway: **New Project → Deploy from GitHub repo** → выбери свой репозиторий
2. После первого деплоя зайди в сервис → вкладка **Settings**:
   - **Networking → Generate Domain** — получишь адрес вида `https://ai-radar-miniapp-production.up.railway.app`
3. Вкладка **Variables** → добавь переменную:
   - `ADMIN_TOKEN` = любая длинная случайная строка (например 30+ символов). Это пароль, по которому утренняя задача будет заливать свежие новости.
4. (Важно для сохранности лайков) Правый клик по сервису → **Attach Volume**, mount path любой, например `/data`. Без volume лайки будут сбрасываться при каждом деплое.

Проверка: открой `https://<твой-домен>/api/health` — должно показать `{"ok":true,"items":25,...}`. Открой сам домен в браузере — увидишь апку.

### Шаг 3. Подключи к Telegram

1. Открой [@BotFather](https://t.me/BotFather)
2. `/newbot` → придумай имя и username бота (например `ai_radar_news_bot`)
3. `/newapp` → выбери своего бота → название «AI-Радар», описание, картинка 640×360 (любая), **Web App URL** = твой Railway-домен, короткое имя, например `radar`
4. Готово: прямая ссылка на апку — `https://t.me/<username_бота>/radar`. Её и раздавай друзьям.
5. Бонус: в BotFather → **Bot Settings → Menu Button** → укажи тот же URL, чтобы апка открывалась кнопкой прямо в чате с ботом.

### Шаг 4. Автообновление новостей

Вернись в чат Cowork и напиши мне:

> Mini App задеплоена, домен https://... , ADMIN_TOKEN: ...

Я добавлю в утреннюю задачу (9:45) шаг, который будет заливать свежие новости прямо в апку. Токен нигде больше не свети.

## Как работают лайки

- 👍/👎 может поставить каждый пользователь (по одному на новость, можно передумать)
- Сервер суммирует голоса всех
- Фильтр «🔥 Топ» сортирует новости по рейтингу, у новостей с рейтингом ≥3 появляется метка «🔥 популярное»

## API (для справки)

- `GET /api/news` — текущие новости
- `GET /api/votes` — суммарные голоса
- `POST /api/vote` — `{uid, id, v: 1|-1|0}`
- `GET /api/ingest?token=...&tag=...&seq=N&total=M&data=<base64url>` — приём свежих новостей кусками (для автоматики)
- `GET /api/health` — статус
